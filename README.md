# رضا لخدمات تفصيل الخزائن والموبيليا | نجار بالرياض 🪚

موقع إلكتروني تعريفي (Landing Page) مصمم خصيصاً ليطابق سياسات وإرشادات جوجل للأنشطة التجارية (Google Business Profile) بنظام نطاق الخدمة (Service Area Business - SAB). الموقع مبرمج ليكون خفيفاً، سريعاً، ومهيأ بقوة لمحركات البحث (SEO).

## 🌟 المميزات (Features)
- **دعم متعدد اللغات:** عربي (RTL)، إنجليزي، وفرنسي (LTR) مع تبديل ديناميكي سلس لاتجاه الصفحة والنصوص.
- **الوضع الليلي والنهاري (Dark/Light Mode):** تصميم مريح للعين يتغير بنقرة زر أو يتعرف تلقائياً على إعدادات نظام المستخدم.
- **متوافق مع سياسات جوجل (SAB):** مدمج بأكواد Schema Markup نوع `ServiceAreaBusiness` مخصصة لمدينة الرياض لضمان قبول الملف التجاري وأرشفته بسرعة.
- **تصميم متجاوب (Responsive):** متوافق 100% مع الجوالات والتابلت والكمبيوتر، مع شريط تنقل سفلي (Bottom Nav) مخصص لتجربة مستخدم مثالية على الموبايل.
- **أزرار عائمة للتواصل:** أزرار اتصال وواتساب جذابة بتأثيرات حركية (Animations) لزيادة معدل التحويل (Conversion Rate).

## 🛠️ التقنيات المستخدمة (Tech Stack)
- HTML5 (Clean & Semantic)
- CSS3 (Vanilla + Custom CSS Variables)
- JavaScript (Vanilla) - بدون أي مكتبات خارجية لضمان أقصى سرعة.
- FontAwesome (Icons)
- Google Fonts (خط تجوال - Tajawal)

## 🚀 الاستضافة والتشغيل (Deployment)
هذا المشروع يتكون من ملف واحد (`index.html`) مصمم ليعمل كـ Single Page Application، وتمت استضافته على خوادم **GitHub Pages** لضمان سرعة الاستجابة، مع ربطه بنطاق (Domain) مخصص من **GoDaddy**.
